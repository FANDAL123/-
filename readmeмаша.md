Работа 2. Линейная и логарифмическая регрессия
Что сделано:
    • Проведён регрессионный анализ зависимости y от x:
        ◦ Линейная модель: Ŷ = a + b·x
        ◦ Логарифмическая модель: Ŷ = a + b·ln(x)
    • Рассчитаны:
        ◦ Коэффициенты регрессии.
        ◦ Остатки, ошибки аппроксимации, R², стандартная ошибка.
    • Сравнительная оценка качества моделей.

Практическое задание 1. Множественная регрессия
Что сделано:
    • Построена модель множественной регрессии: y = a + b₁·x₁ + b₂·x₂.
    • Рассчитаны:
        ◦ Коэффициенты регрессии, стандартизованные коэффициенты.
        ◦ Коэффициенты эластичности, множественный коэффициент корреляции R.
        ◦ Частные коэффициенты корреляции.
    • Проведена проверка значимости модели с помощью F-критерия.
